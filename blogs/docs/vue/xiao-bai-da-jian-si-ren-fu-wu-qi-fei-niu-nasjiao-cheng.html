电脑小白也可以搭建私人服务器-飞牛nas教程

Sakura 2025-06-03 飞牛nas服务器系统安装

介绍
电脑小白也可以搭建一台属于自己的私人服务器，

搭建私人服务器-飞牛nas教程 存储备份手机照片和视频

# 🎏 使用场景

# 1、准备一个16g的U盘， nas主机插上网线

# 2、飞牛私有云官网 https://www.fnnas.com/ 下载最新的iso 镜像

# 3、下载rufus 写盘工具， 将文件烧录到U盘里面，推荐写盘工具可选择 Rufus，下载地址：https://rufus.ie/zh/

# 4、打开rufus 选择对应的u盘， 选上刚下载的 飞牛 iso 镜像文件， 点击准备就绪， 点击开始， 完成后拔出u盘

# 5、将制作好的u盘插到 要用做服务器的主机上， 点击对应的主板BIOS 热键 选择u盘启动， 选择Graphical install 图形化安装流程， 直接回车， 如果失败， 那么就回到bios 界面 关闭Secure Boot重试

# 6、 如果还是安装失败， 可以选择 Rescuing Install 应急模式安装 重试

# 7、 回车完后， 会弹出一个选择框， 选择要安装的位置硬盘， 并设置分区， 系统建议设置为64g ，点击下一步，安装完后， 选择对应的网卡，也就是网线 获取动态ip， 确定，然后自动重启系统

# 8、 重启系统，直到黑框框的界面弹出 大大的 FNOS 的字符， 并且下面有一个ip 地址， 一般为 http:192.168.*.**:5666,

# 9、在自己的电脑浏览器上输入这个ip地址，注册和登录， 并创建对应的存储空间

# 10、 远程访问， 在系统设置中，选择远程访问， 需要登录飞牛的账号， 如果不想用飞牛自带的远程访问， 可以设置ipv6的方式， 需要网线支持ipv6，和路由支持ipv6 设置和防火墙开关， 并使用路由拨号模式
